ILO #3 Skilled Collaborators

Use Github to build a website together.

Learn the industry practice of using version control and merging branches.

How did we accomplish this goal?

We had a shared github repository where we would take branches off from the master, write our code, and merge it into the master branch.

Takeaways:

Communication is key to collaboration. If we didn't have specific things to be working on, we would often have conflicts in our code that were a mess to sort out. When we had specific files that were our own responsibility, we were able to work at full speed without conflicts.
