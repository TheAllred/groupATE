ILO #4 Sound Thinkers

Build a simple game in a new programming language

Demonstrate our ability to learn and help each other by learning the basics of a new language well enough to make a game.

How did we accomplish this goal?

We each chose a new language that we didn't know to learn and we shared our game and it's code with our teammates.

Takeaways:

Focus on what you do know. A lot of these programming languages are very similar in function and in syntax. When we focused on the similarities to the things we already knew, learning the new language was easy. But if we made it too complicated, we would get stuck.
